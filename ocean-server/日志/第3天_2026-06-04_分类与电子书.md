## 二、技术实现要点

### 分类树形结构
```java
// 查询所有分类后，在内存中构建树形结构
List<Category> allCategories = categoryMapper.selectList(null);
// 递归设置子分类
private void buildTree(List<Category> categories, Category parent) {
    List<Category> children = categories.stream()
        .filter(c -> c.getParent().equals(parent.getId()))
        .collect(Collectors.toList());
    parent.setChildren(children);
    children.forEach(child -> buildTree(categories, child));
}
```
