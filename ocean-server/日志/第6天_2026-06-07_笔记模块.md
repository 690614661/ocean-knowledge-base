### 笔记查询权限控制
```java
public PageResp<Note> listNotes(Integer pageNum, Integer pageSize, Long userId) {
    Page<Note> page = new Page<>(pageNum, pageSize);
    
    QueryWrapper<Note> wrapper = new QueryWrapper<>();
    // 只查询公开笔记或当前用户的笔记
    wrapper.and(w -> w.eq("is_public", 1)
                      .or()
                      .eq("user_id", userId));
    wrapper.orderByDesc("create_time");
    
    Page<Note> result = noteMapper.selectPage(page, wrapper);
    return new PageResp<>(result.getTotal(), result.getRecords());
}
```

### 笔记点赞
```java
@Transactional
public void voteNote(Long noteId, Long userId) {
    Note note = noteMapper.selectById(noteId);
    if (note == null) {
        throw new BusinessException("笔记不存在");
    }
    
    // 检查是否已点赞
    String key = "note:vote:" + noteId + ":" + userId;
    if (redisTemplate.hasKey(key)) {
        throw new BusinessException("请勿重复点赞");
    }
    
    // 更新点赞数
    note.setVoteCount(note.getVoteCount() + 1);
    noteMapper.updateById(note);
    
    // 记录点赞状态
    redisTemplate.opsForValue().set(key, "1", 1, TimeUnit.DAYS);
}
```
