  const record = await axios.post(`/quiz/start/${quizId}`);
  const questions = await axios.get(`/quiz/questions/${record.data.id}`);
  this.questions = questions.data.content;
  this.recordId = record.data.id;
}

## 五、遇到的问题与解决方案

**问题1**：前后端联调时跨域问题  
**解决**：配置CorsConfig，允许前端域名跨域访问

**问题2**：答题页面加载慢  
**解决**：使用分页加载题目，避免一次性加载所有题目

**问题3**：个人中心数据展示不完整  
**解决**：使用关联查询，补充用户信息、答题统计等数据
