### 文档树形结构
```java
// 查询指定电子书下的所有文档
List<Doc> docs = docMapper.selectList(
    new QueryWrapper<Doc>()
        .eq("ebook_id", ebookId)
        .orderByAsc("sort")
);
// 构建树形结构
buildDocTree(docs, parentDoc);
```

### 文档与内容关联保存
```java
@Transactional
public void saveDoc(DocSaveReq req) {
    // 保存文档基本信息
    docMapper.insert(doc);
    // 保存文档内容
    Content content = new Content();
    content.setId(doc.getId());
    content.setContent(req.getContent());
    contentMapper.insert(content);
}
```

### 点赞防重复
```java
// 使用Redis记录用户点赞状态
String key = "doc:vote:" + docId + ":" + userId;
if (redisTemplate.hasKey(key)) {
    throw new BusinessException("请勿重复点赞");
}
redisTemplate.opsForValue().set(key, "1", 1, TimeUnit.DAYS);
```

## 五、遇到的问题与解决方案

**问题1**：文档树形结构排序不正确  
**解决**：在查询时添加orderByAsc("sort")，确保按排序值升序排列

**问题2**：文档和内容分开存储导致事务问题  
**解决**：使用@Transactional注解确保数据一致性，失败时回滚

**问题3**：富文本内容过大影响查询性能  
**解决**：将文档基本信息和内容分表存储，查询列表时不加载内容
