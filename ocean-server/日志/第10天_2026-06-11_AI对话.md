## 二、技术实现要点

### AI Provider接口
```java
public interface AiProvider {
    AiResponse chat(AiRequest request);
}

@Component
public class DeepSeekProvider implements AiProvider {
    
    @Value("${ai.deepseek.api-key}")
    private String apiKey;
    
    @Value("${ai.deepseek.api-url}")
    private String apiUrl;
    
    @Override
    public AiResponse chat(AiRequest request) {
        OkHttpClient client = new OkHttpClient();
        
        // 构建请求体
        JsonObject body = new JsonObject();
        body.addProperty("model", "deepseek-chat");
        body.add("messages", convertMessages(request.getMessages()));
        
        Request httpRequest = new Request.Builder()
            .url(apiUrl)
            .header("Authorization", "Bearer " + apiKey)
            .post(RequestBody.create(body.toString(), MediaType.parse("application/json")))
            .build();
        
        // 发送请求
        Response response = client.newCall(httpRequest).execute();
        return parseResponse(response);
    }
}
```

### 对话上下文管理
```java
public AiResponse chat(Long userId, String conversationId, String userMessage) {
    // 获取历史消息
    List<AiMessage> history = aiMessageMapper.selectList(
        new QueryWrapper<AiMessage>()
            .eq("conversation_id", conversationId)
            .orderByAsc("create_time")
            .last("LIMIT 10") // 保留最近10条消息
    );
    
    // 构建请求
    AiRequest request = new AiRequest();
    List<ChatMessage> messages = new ArrayList<>();
    history.forEach(msg -> {
        messages.add(new ChatMessage(msg.getRole(), msg.getContent()));
    });
    messages.add(new ChatMessage("user", userMessage));
    request.setMessages(messages);
    
    // 调用AI
    AiResponse response = aiProvider.chat(request);
    
    // 保存消息
    saveMessage(conversationId, "user", userMessage, 0, 0);
    saveMessage(conversationId, "assistant", response.getContent(), 
                response.getPromptTokens(), response.getCompletionTokens());
    
    // 记录用量
    logUsage(userId, response);
    
    return response;
}
```
